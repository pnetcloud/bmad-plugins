# Docusaurus Standards

- Organize docs into clear sections with sidebar structure.
- Use `docs/` folder with `sidebars.js` or `sidebars.ts`.
- Write in Markdown/MDX; keep docs concise and structured.
- Apply SEO tags and metadata for each page.
- Version documentation for major releases.
- Deploy docs via CI/CD pipeline; validate links and build.
